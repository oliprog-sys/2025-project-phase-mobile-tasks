
# A simple E-Commerce Mobile App (Flutter)
  An e-commerce mobile app that allows user to add, search, view details or see all the products.
  It is for mobile development task and way of learning different things like understanding flutter basics, navigation and routing and simple CRUD operation
  
## Features
  ### Home Page
    - list of all products
    - image and small information of each product in each card
    - clickable cards to see the product detail
  ### Search Page
    - search using product name or category name
    - filter through price and category dropdown
  ### Add product page
      #### Add the product details like
          - name
          - category
          - price
          - descriptino
### Product Details
  - show complete information about the selected product
  - has update and delete feature of a specific product
## How to run the App
  ### clone the repository
    - git clone [GitHub Repository](https://github.com/oliprog-sys/2025-project-phase-mobile-tasks/tree/main/on-boarding/product_3)
    - cd product_3
  ### Install dependencies
      - flutter pub get
  ### Run the App
      - flutter run
